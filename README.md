# Bank Account TDD (C#)

This project demonstrates Test-Driven Development (TDD) using C# and MSTest.

## Overview
A simple bank account system that supports debit and credit operations with proper validation and exception handling.

## Features
- Debit functionality with validation
- Credit functionality with validation
- Exception handling for invalid operations
- Unit tests for all major scenarios

## Test Cases
- Debit with valid amount
- Debit with negative amount (throws exception)
- Debit exceeding balance (throws exception)
- Credit with valid amount
- Credit with negative amount (throws exception)

## Technologies Used
- C#
- .NET
- MSTest
- Visual Studio 2022

## Purpose
This project was created as part of COMP2151 Lab 8 to practice Test-Driven Development (TDD) principles.
